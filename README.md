# Movie_-_TV_Show_Metadata_Analysis
This project explores patterns in movie and TV show metadata, analyzing genre trends, director influence, certificate impact, and more.
# Tools
- Python (Pandas, Seaborn, Matplotlib)
- Jupyter Notebook
- PostgreSQL
- Excel
- Power BI
